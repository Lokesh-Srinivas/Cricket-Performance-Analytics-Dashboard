# Cricket Performance Analytics Dashboard

## Overview
End-to-end data analytics project analyzing 7,500+ international cricket 
player records across ODI, T20I and Test formats.

## Tech Stack
- Python (Pandas) — Data cleaning & transformation
- PostgreSQL — Data warehouse & SQL analysis
- Tableau — Interactive dashboard
- GitHub — Version control

## Project Structure
- `cricket_analysis.ipynb` — Python data cleaning & exploration
- `cricket_queries.sql` — 10 SQL queries for performance analysis
- `Batting/` — Raw batting data (ODI, T20, Test)
- `Bowling/` — Raw bowling data
- `Fielding/` — Raw fielding data
- `cricket_combined.csv` — Clean combined dataset (7,500 records)

## Key Insights
- Tendulkar leads ODI run scorers with 18,426 runs in 463 matches
- Bradman's Test average of 99.94 is 37 points ahead of the next best
- Only 12 players appear in top 20 of both ODI and Test run scorers
- Test cricket has the highest batting average (20.56) vs T20 (15.47)
